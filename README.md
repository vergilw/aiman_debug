### ReactNative配置

参考官方配置方案 https://reactnative.cn/docs/getting-started/

1. 确保WindowsOS下已安装如下，且环境变量配置正确。
Node 10.x版本+
Python2
JavaSDK 1.8

yarn
react-native-cli

AndroidStudio IDE

2. 环境配置完成后，用终端进入项目主目录，
执行 yarn install
执行 ./node_modules/.bin/rn-nodeify --hack --install
打开Android模拟器
执行react-native run-android
